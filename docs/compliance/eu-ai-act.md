# EU AI Act Evidence

CapFence can generate evidence packs that help teams document runtime controls for high-risk AI systems.

```bash
capfence eu-ai-act ./src --output eu-ai-act-report.html
```

## Evidence areas

CapFence can provide evidence for:

- runtime authorization controls
- policy configuration
- tool gating coverage
- audit logging
- replayability
- approval workflows
- risk assessment reports

## Practical use

Use the generated report as a starting point for Annex IV-style documentation. Review it with legal, compliance, and security teams before relying on it for regulatory submissions.

## Important note

CapFence provides technical evidence. It does not determine legal classification, regulatory obligations, or final compliance status.
