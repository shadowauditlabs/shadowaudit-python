# Demo Cast

This is a short terminal cast of the CapFence demo run. If you have `asciinema` installed, you can record a fresh cast with the steps below.

## Record

```bash
asciinema rec docs/examples/demo.cast
./scripts/demo.sh
```

## Play

```bash
asciinema play docs/examples/demo.cast
```

## Notes

- Keep the demo under 2 minutes.
- Use the default terminal font and size for readability.
- Re-record if output changes significantly.
